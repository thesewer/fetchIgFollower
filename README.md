# fetchFollower

### install

* ssh

```sh
git clone git@github.com:thesewer/fetchFollower.git
```

* gh
  
```sh
gh repo clone thesewer/fetchFollower
```

* https (CURL)

```sh
git clone --depth=1 https://github.com/thesewer/fetchFollower.git ~/fetchFollower
```

## Usage

```sh
cd ~/fetchFollower
npm start
```
